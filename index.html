📖 О языке КРС

КРС (Cursed Coding Style) — это язык разметки, который превращает простые теги в полноценные HTML, CSS и JavaScript.

🔤 ВАЖНОЕ ПРАВИЛО:
Каждая команда открывается <c.имя_команды> и закрывается <cc.имя_команды>.
НИКОГДА не используйте </c.имя> — только <cc.имя>!
⚙️ Как это работает:
1. Весь код пишется внутри <CURSED-CODING> ... <CCURSED-CODING>
2. Парсер превращает <c.тег> в реальные HTML-элементы
3. У тегов есть атрибуты id и class
4. <c.style> — для КРС-КСС (русские стили)
5. <c.script> — для КРС-ЖС (русские скрипты)

🎨 КРС-КСС (русские стили)

Пишите стили на русском языке внутри <c.style> ... <cc.style>

кнопка {
  фон = синий
  цвет = белый
  отступ = 12px 24px
  углы = 30px
  тень = 0 4px 12px серый
}

кнопка при наведении {
  фон = тёмно-синий
}

карточка {
  фон = белый
  тень = 0 2px 8px серый
  углы = 12px
}
<cc.style>

⚡ КРС-ЖС (русские скрипты)

Пишите логику на русском языке внутри <c.script> ... <cc.script>

<c.script>
let счет = 0;

кнопка {
  при нажатии показать('Счёт: ' + счет)
  счет = счет + 1
}

поле {
  при вводе показать('Вы ввели: ' + значение)
}
<cc.script>
🧠 Что работает в КРС-ЖС:
— кнопка { при нажатии показать('текст') }
— поле { при вводе показать('Ввели: ' + значение) }
— карточка { при наведении показать('На карточке') }
— Переменные let имя = значение
— показать('текст' + переменная)

⚡ Правила синтаксиса
  • 1. Обязательная обёртка — весь код внутри <CURSED-CODING> ... <CCURSED-CODING>
  • 2. Закрывающие теги — строго <cc.имя_команды>, НЕ </c.имя>
  • 3. Атрибуты — работают id="..." и class="..."
  • 4. Вложенность — команды можно вкладывать друг в друга
  • 5. КРС-КСС — русские свойства: фон = синий, углы = 8px
  • 6. КРС-ЖС — русские события: при нажатии, при вводе, при наведении
