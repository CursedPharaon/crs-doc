📖 О языке CRS

CRS (Cursed Coding Style) — это язык разметки, который превращает простые теги в полноценные HTML, CSS и JavaScript. Команды можно писать на английском, а стили и скрипты — на русском (КРС-КСС и КРС-ЖС).

🔤 ВАЖНОЕ ПРАВИЛО:
Каждая команда открывается <c.имя_команды> и закрывается <cc.имя_команды>.
НИКОГДА не используйте </c.имя> — только <cc.имя>!
⚙️ Как это работает:
1. Весь код пишется внутри <CURSED-CODING> ... <CCURSED-CODING>
2. Парсер превращает <c.тег> в реальные HTML-элементы
3. <c.style> и <c.script> позволяют добавлять CSS и JS

🐄 КРС-КСС и КРС-ЖС — русские стили и скрипты

🎨 КРС-КСС (Каскадные Русские Стандартные стили)

Пишите стили на русском языке внутри <c.style> ... <cc.style>

<c.style>
кнопка {
  фон = зелёный
  цвет = белый
  отступ = 8px 16px
  углы = 8px
}

кнопка при наведении {
  фон = тёмно-зелёный
}

поле {
  отступ = 8px
  ширина = 200px
  углы = 6px
  рамка = 1px сплошной серый
}

карточка {
  фон = белый
  тень = 0 4px 12px серый
  углы = 14px
}
<cc.style>

⚡ КРС-ЖС (Живой Сценарий на русском)

Пишите логику на русском языке внутри <c.script> ... <cc.script>

<c.script>
кнопка {
  при нажатии показать('Привет, мир!')
}

поле {
  при вводе показать('Ввели: ' + значение)
}

кнопка {
  при нажатии показать('Текст на кнопке: ' + текст)
}
<cc.script>
🧠 Как это работает:
— кнопка → превращается в .crs-btn
— поле → превращается в .crs-input
— карточка → превращается в .crs-card
— показать() → превращается в alert()
— значение → превращается в this.value
— текст → превращается в this.innerText

⚡ Правила синтаксиса
  • 1. Обязательная обёртка — весь код должен быть внутри <CURSED-CODING> ... <CCURSED-CODING>
  • 2. Закрывающие теги — строго <cc.имя_команды>, НЕ </c.имя> и НЕ </cc.имя>
  • 3. Регистр — все команды пишутся нижним регистром (c.button, c.text)
  • 4. Вложенность — команды можно вкладывать друг в друга
  • 5. КРС-КСС — внутри <c.style> используйте русские свойства: фон = зелёный, углы = 8px
  • 6. КРС-ЖС — внутри <c.script> используйте: кнопка { при нажатии показать('текст') }
  • 7. Селекторы в КРС-ЖС: кнопка, поле, карточка, текст, заголовок
💡 Самые частые ошибки:
— Использование </c.tag> вместо <cc.tag>
— Забытый закрывающий тег <cc.tag>
— Русские команды в открытии тега (пишем c.button, а не c.кнопка)
🚀 Быстрый старт за 2 минуты
1. Пишем каркас:
<CURSED-CODING>
  ... твой код ...
<CCURSED-CODING>

2. Добавляем заголовок:
<c.title> Моя страница <cc.title>

3. Текст и кнопка:
<c.text> Привет! <cc.text>
<c.button> Нажми <cc.button>

4. Добавляем стиль на русском:
<c.style>
кнопка {
  фон = зелёный
  углы = 10px
}
<cc.style>

5. Добавляем поведение на русском:
<c.script>
кнопка {
  при нажатии показать('Привет!')
}
<cc.script>

6. Готово! Запускаем.
